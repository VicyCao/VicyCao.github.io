R code：二阶段设计的样本量计算

2021-02-12

由于可以提前因无效终止试验，Simon二阶段设计很受临床医生的欢迎，如何在开源软件R语言中计算Simon二阶段设计的样本量呢？

R语言因开源、灵活得以在临床试验中广泛使用，其中最突出的优点就是开发者可以上传自己写的包来实现特定问题的计算。

Simon二阶段设计的样本量计算同样可以在R语言中轻松实现：
R Code如下：

install.packages("ph2mult")    #首先安装R包ph2mult
library(ph2mult)                      #加载包
binom.design(type = "admissible", p0 = 0.05, p1 = 0.15, 
             signif.level = 0.05, power.level = 0.8,
             plot.out = TRUE)

其中：
type = "admissible"：列出推荐使用的样本量计算结果
p0 = 0.05：历史参照率值（二分类结局指标，如ORR）
p1 = 0.15：预估本试验的率值
signif.level = 0.05：单侧α为0.05（Simon二阶段设计默认为单侧检验）
power.level = 0.8：检验效能为0.8，对应β为0.2
plot.out = TRUE：生成以总样本量为横坐标、期望样本量为纵坐标的折线图
