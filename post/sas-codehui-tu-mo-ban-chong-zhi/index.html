2021-02-09

关注一个SAS绘图模板重置的小问题

接触过SAS的童鞋都知道，SAS自带的绘图效果极丑（这里指不经proc sgplot或Macro修饰过的原图），具体感受可通过观看以下生存曲线图像获得：

🤣🤣🤣🤣🤣🤣🤣🤣🤣🤣🤣🤣🤣🤣🤣

有时为了做大会的壁报或者文章投稿需要，会编写SAS的Macro重新绘制颜值较高的生存曲线图像，此时会发现后续只要跑proc lifetest程序，SAS的HTML输出页面就都是Macro设置的图像样式，这时候想查看SAS原始自带的图像就有点犯难，毕竟有些奇葩杂志会要求提供未经修饰的SAS原图作为投稿的补充材料。

这个时候部门的MA小伙伴就满眼期待地望着我，希望我能帮忙提供原始图像。在我第一次遇到MA小伙伴求助此类问题时，我的内心是慌张的，毕竟我都跑过那么多次Macro了，原始图像的样式早就模糊了，也不知道从哪里再倒腾出来，但还是要干啊，通过查资料，发现几行代码就可以解决这个小问题：

SAS code如下：

ods path sashelp.tmplmst (read);
proc datasets library=sasuser nolist;
    delete templat (memtype=itemstor);
run;
ods path reset;

虽然是个小问题，但是觉得还是值得记录一下的，SAS大佬请忽略😂
